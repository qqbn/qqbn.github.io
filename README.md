# My own portfolio page 🚀

Personal portfolio site built with Astro. Light/dark theme, animated project slider, scroll-driven animations, and a contact section with clipboard copy support.

🌐 Live: [qqbn.github.io](https://qqbn.github.io)

## 🛠️ Tech stack

- [Astro 4](https://astro.build) — static site generator
- [TypeScript](https://www.typescriptlang.org) — type safety across components
- [AOS](https://michalsnik.github.io/aos/) — scroll-driven animations
- Canvas 2D — interactive hero visual
- Astro Content Collections — projects, experience, and contact data as Markdown files
- GitHub Pages + GitHub Actions — hosting and CI/CD

## ⚡ Getting started

```bash
npm install
npm run dev      # dev server at localhost:4321
npm run build    # production build → dist/
npm run preview  # preview the build locally
```
