# Portfolio

My personal portfolio — a site showcasing my experience, tech stack, and projects as a Senior Frontend Developer.

## Tech stack

- **Astro.js** — framework
- **Vanilla CSS** — styling
- **AOS (Animate On Scroll)** — scroll animations
- **Canvas 2D** — interactive shape in the hero section
- **Astro Content Collections** — project content (Markdown)
- **GitHub Pages + GitHub Actions** — hosting and CI/CD

## Local development

```bash
npm install
npm run dev
```
